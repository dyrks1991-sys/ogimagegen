# OGImageGen

> DevOS project initialized 2026-07-07

## Overview

Open Graph image generator — paste a title, get a shareable card in seconds.

## Quick Start

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # production build → out/
```

## Stack

- **Next.js 14** — static export (`output: 'export'`)
- **TypeScript** — strict mode
- **Tailwind CSS 3**

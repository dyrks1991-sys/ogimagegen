## Risks Acknowledged ✓

- [x] **R1:** Canvas text rendering differs between browsers (font hinting, subpixel) → Mitigation: Use system-ui font stack in canvas. Test on Chrome, Firefox, Safari before release.
- [x] **R2:** Long title text overflows canvas horizontally → Mitigation: Implement wrapText() in ogRenderer.ts — splits text into lines that fit within maxWidth, respects 2-line maximum for title.
- [x] **R3:** canvas.toBlob() null callback in Safari → Mitigation: Wrap in Promise with null check. Fallback: canvas.toDataURL("image/png") → link download.
- [x] **R4:** Canvas export is 72dpi (appears low-res on Retina) → Mitigation: OG images are displayed at 600px max — 1200px canvas at 1x is already 2x retina. Document this.
- [x] **R5:** Color picker native styling differs significantly across browsers/OS → Mitigation: Use native <input type="color"> wrapped with custom label. Acceptable UX variance — functional on all platforms.

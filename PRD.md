## Vision

Make every developer's blog post, product launch, and side project look professionally branded in 60 seconds — no design skills, no Figma, no paid tools.

## Description

A fully client-side Open Graph image generator. Users enter a title and description, pick one of three templates, choose brand colors, and see a live 1200×630 canvas preview. One click downloads a production-ready PNG — no uploads, no accounts, no servers. All rendering happens in the browser via the Canvas 2D API.

## MVP Features

| ID | Feature | Priority | Acceptance Criteria |
|---|---|---|---|
| F1 | **Title Input** | P0 | Text field, max 80 chars. Character counter shown. Updates canvas within 50ms. |
| F2 | **Description Input** | P0 | Text field, max 120 chars. Optional — shows default if empty. Updates canvas within 50ms. |
| F3 | **Template Selector** | P0 | 3 templates: Dark (default), Light, Gradient. Each has a mini-preview thumbnail. Switching updates canvas immediately. |
| F4 | **Color Customization** | P0 | Background color + accent color pickers using <input type="color">. Hex value shown. Resets to template defaults on template switch. |
| F5 | **Live Canvas Preview** | P0 | 1200×630 canvas scaled to ~600×315px for display. Rerenders on every config change. No blank state — default content pre-populated. |
| F6 | **PNG Download** | P0 | canvas.toBlob("image/png") → file download. Filename: og-YYYYMMDD-HHMMSS.png. Button always enabled (canvas always has content). |
